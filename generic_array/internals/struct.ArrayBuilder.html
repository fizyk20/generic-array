pub struct ArrayBuilder<T, N: ArrayLength> { /* private fields */ }
Expand description

UNSAFE: Creates an array one element at a time using a mutable iterator of pointers.

You MUST increment the position while iterating to mark off created elements, which will be dropped if into_inner is not called.

pub unsafe fn iter_position( &mut self ) -> (IterMut<'_, MaybeUninit<T>>, &mut usize)

Creates a mutable iterator for writing to the array elements.

You MUST increment the position value (given as a mutable reference) as you iterate to mark how many elements have been created.

#[cfg(feature = "internals")]
fn make_some_struct() -> SomeType { SomeType }
unsafe {
    let mut builder = ArrayBuilder::<SomeType, U5>::new();
    let (dst_iter, position) = builder.iter_position();
    for dst in dst_iter {
        dst.write(make_some_struct());
        // MUST be done AFTER ownership of the value has been given to `dst.write`
        *position += 1;
    }
    let your_array = builder.assume_init();
}
